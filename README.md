# Expense Tracker

A simple application to track your expenses, configure currency preferences, and visualize spending with graphs

## Planned Features

### Add Expense Entries
- A GUI to add spending entry of details like amount, timestamp, category and description

### Configure Currency
- Users may configure displaying currency

## Graphs
- Graphs to analyze spending patterns over time
  - Monthly summaries  
  - Category breakdowns
